# 제로베이스 프론트엔드 스쿨_SquareRoot

---

### 문제 설명
정수값 n이 주어졌을 때, 이 값의 제곱근을 출력하는 프로그램을 구현하세요.
단, n의 제곱근이 무리수일 경우 0을 반환합니다.

### 입력 형식
n: 정수

### 출력 형식
n의 제곱근을 정수로 반환

### 제약 사항
1 <= n <= 1000000

### 입출력 예시

---

#### 예시1
#### 입력 : n = 4
#### 출력 : 2
#### 설명
4의 제곱근은 2이므로, 2를 반환

---

#### 예시2
#### 입력 : n = 6
#### 출력 : 0
#### 설명
6의 제곱근은 무리수이므로, 0을 반환

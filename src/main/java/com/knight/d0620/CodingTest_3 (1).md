# 제로베이스 프론트엔드 스쿨_GreatestCommonFactor

---

### 문제 설명
정수 배열 A에는 0보다 큰 숫자가 N개 들어있습니다.
N개의 모든 숫자를 아우르는 최대 공약수를 구하는 프로그램을 구현하세요.

### 입력 형식
A: 자연수가 담긴 정수 배열

### 출력 형식
최대 공약수를 정수로 반환

### 제약 사항
0 < A.length <= 1000
0 < A[i] <= 100

### 입출력 예시

---

#### 입력 : A = {6, 12, 4}
#### 출력 : 2
#### 설명
모든 세 숫자의 최대 공약수는 2이다.

# 제로베이스 프론트엔드 스쿨_Calculator

---

### 문제 설명
문자열 계산식을 입력받는 계산기를 만들려고 합니다.
계산식은 덧셈(+), 뺄셈(-), 숫자로만 이루어져 있습니다.
계산식 s가 주어질 때, 계산식의 계산 결과를 출력하는 프로그램을 구현하세요.

### 입력 형식
s: 숫자와 덧셈, 뺄셈으로 이루어진 계산식 문자열

### 출력 형식
계산한 결과를 정수로 반환

### 제약 사항
0 < s.length <= 10

### 입출력 예시

---

#### 입력 : s = "-3+26-7"
#### 출력: 16
#### 설명
위 수식의 계산 결과는 16이다.
